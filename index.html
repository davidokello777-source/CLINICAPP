📲 Install TALI MEDICALS

🤖 Android (Chrome)
  1. Tap the ⋮ Menu (top right)
  2. Select "Add to Home screen"
  3. Tap "Add" to confirm
💻 PC / Desktop
🟢 Google Chrome
  1. Open the site in Google Chrome
  2. Look for the ⊕ install icon at the far right of the address bar
  3. Click it → a popup says "Install TALI MEDICALS?"
  4. Click the blue "Install" button
  5. The app opens in its own window — shortcut added to Desktop & Start Menu
💡 If no icon shows: Chrome Menu (⋮) → "Install TALI MEDICALS…"
🔵 Microsoft Edge
  1. Open the site in Microsoft Edge
  2. Look for the ⊕ install icon in the address bar (right side)
  3. Click it → popup: "Install this site as an app?"
  4. Click "Install"
  5. App is pinned to Taskbar & Start Menu automatically
💡 Or: Edge Menu (…) → Apps → "Install this site as an app"
⚠️ Firefox / Safari (Desktop)
Firefox & Safari desktop do not support PWA install.
Use Chrome or Edge on PC for the best experience.
🍎 iPhone / iPad (Safari)
  1. Tap the Share button (□↑)
  2. Scroll down, tap "Add to Home Screen"
  3. Tap "Add" to confirm
